Preferencias generales

El idioma del programa 3 se puede cambiar arriba a la izquierda. El cambio afectará casi todo el texto del programa. Deberás reiniciar BlueBrick para que los cambios tengan efecto. Los elementos que no e verán afectados por el cambio son las pestañas encima del área de piezas y ventanas que están relacionadas directamente con el idioma de tu sistema operativo (por ejemplo la paleta de selección colores).

The Notifications 4 list some warning messages that can popup during the execution of BlueBrick. Since those warning messages can be repetitive, you can disable them by checking a "forget" checkbox. By default all the warning notifications are enabled. You can re-enable the notification messages you have chosen to ignore, or disable some notifications upfront.

La lista de archivos recientes 5 tiene por defecto 4 entradas. El máximo son 20. Puedes borrar la lista usando el botón "Borrar lista de archivos recientes". Esta acción no se puede deshacer.

La lista de deshacer/rehacer 6 se puede ajustar. El número máximo de acciones a deshacer es de 1000. El número máximo de acciones mostradas es de 50. Nota que cuanto mayor sea el número de acciones a deshacer que se graban, más memoria ocupará el programa.

The New Map Template field 7 let you specify a BBM file that should be used as a template whenever you create a new layout. To do that, first create a BBM file with the options you prefer, like the number and type of layers, the background color, your name as author, the logo of your LUG in one corner, etc... and save it somewhere on your hard drive. Then specify the path of that file in this field. Now on, every time you use the  File > New  feature, a new layout will be created by using this file as the starting point.

Los ajustes de rendimiento 8 pueden ser importantes para ordenadores más viejos. Por defecto está seleccionada la opción "más rápido". Puedes seleccionar la opción "Menos memoria" si notas que el sistema se hace inestable durante su uso. La mayoría de ordenadores deberían poder usar los modos "equilibrado" o más rápido"
