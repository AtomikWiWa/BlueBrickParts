Cómo personalizar las teclas de atajo

El botón 1 le permite restaurar todas las preferencias a sus valores por defecto, como cuando instaló BlueBrick por primera vez. El botón 2 le permite restaurar sólo las preferencias mostradas en la categoría visible actual.

Se pueden añadir atajos de teclado utilizando el menú desplegable 4 al final de la lista de atajos de teclado (fig. 9c) Elige las teclas del teclado a las que deseas asignar una función 3, luego la función que esta tecla debe cumplir 4, la pieza o el número de set que usará 5, y el tipo de conexión que tendrá 6. Parta completar el proceso pulsa el botón "Añadir atajo de teclado" 7.

Nota que un atajo de teclado no se puede editar una vez creado. Debes borrar el atajo si deseas cambiarlo y crear uno nuevo. Para eliminar un atajo, selecciónelo y presione el botón "Eliminar el atajo" 8. También puedes eliminar o reasignar los atajos predeterminados si no te gustan.

Estos cambios serán permanentes una vez que pulses el botón "OK". Si pulsas "Cancelar" los cambias se descartarán.
