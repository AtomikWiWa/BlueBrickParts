Áreas de la ventana

La ventana de BlueBrick tiene 3 áreas: el área de trazado, el área de piezas y el área de datos del mapa. El uso de estas áreas se explica con más detalle en la sección Crear y modificar modelos.

El área de trazado

El área de trazado ocupa la parte izquierda de la pantalla y se reconoce por su fondo azul con cuadrícula blanca (por defecto). Este área tiene una herramienta modificable de "ajuste a la cuadrícula" que te ayudará a ajustar diferentes elementos en tu trazado.

El área de piezas

El área de piezas se encuentra en la esquina superior derecha de la ventana principal y consiste en varias pestañas de categorías en las que las piezas están organizadas. Si deseas contribuir a la lista de piezas por favor pulsa aquí. En la actualidad la pestaña "Custom" contiene 3 diferentes tamaños de mesas. Los tamaños representan el número de baseplates de 32x32 que caben en la superficie.

El área de datos del mapa

Este panel está situado en la esquina inferior derecha y tiene tres pestañas. La primera pestaña muestra las capas, la segunda pestaña muestra el uso de las piezas y la última pestaña muestra alguna información general sobre el mapa. La función de capas es especialmente útil para diseños más complicados. En esta pestaña se pueden añadir, eliminar y organizar las distintas capas que se necesiten. Los detalles se pueden encontrar en la sección Creación y uso de capas. La pestaña de uso de las partes muestra la lista de todas las partes utilizadas en el mapa, y eventualmente el porcentaje de uso en relación con el presupuesto. La última pestaña le permite añadir alguna información relativa a su mapa.
