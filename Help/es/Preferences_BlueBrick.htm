BlueBrick Preferences

BlueBrick has a lot of customizable options that you can tune through   edit > Preferences... . These options are grouped in five categories: General Preferences, Edition Preferences, Appearance Preferences, Part Library Preferences and Shortcut Keys.

The button 1 let you restore all the preferences to their default values, as when you installed BlueBrick for the first time. The button 2 let you restore only the preferences displayed on the current visible category.

Some spare options are also available through different ways, you can learn more about it in the Other Options section.
