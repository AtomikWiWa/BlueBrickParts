Seleccionar piezas

Selección general

La selección general funciona del mismo modo en las capas de ladrillos y en las de texto. Puedes seleccionar piezas individuales haciendo clic sobre ellas, o múltiples piezas arrastrando el puntero sobre las piezas que deseas (creando un rectángulo de selección). La forma de destacar las piezas seleccionadas se puede modificar en la pestaña "Apariencia" en Opciones globales.

Se pueden seleccionar múltiples piezas manteniendo pulsada la tecla CTRL (por defecto), mientras se va haciendo clic sobre las piezas con el botón izquierdo del ratón. Para deseleccionar una pieza, hay que mantener pulsada la tecla CTRL y hacer clic sobre ella.

También se puede seleccionar un grupo de piezas pulsando el botón izquierdo del ratón y arrastrándolo para que las piezas deseadas queden dentro del rectángulo de selección. Puedes hacer múltiples selecciones de grupos mientras si se mantiene pulsada la tecla CTRL (por defecto) al crear rectángulos de selección adicionales. Si no se mantiene pulsada la tecla CTRL el rectángulo de selección anterior desaparecerá. Si dos rectángulos de selección que no contienen las mismas piezas se solapan, todas las piezas se unirán en un único rectángulo de selección. Si se arrastra un rectángulo de selección sobre piezas que ya están seleccionadas las deseleccionará.

Path Selection for Brick only

You can also select a series of connected parts, like for example a chunk of rail tracks or road baseplates, by first selecting one part, then hold SHIFT+CTRL (by default) and click on a second part. All the parts between the two selected parts will be selected using the shortest way. If no path can be found between the two initially selected parts, nothing happen. Alternatively, you can first select the two extremities, then right-click and choose Select Path in the contextual menu or press CTRL+P.

Note that the path will always start from the last part selected (i.e. the last part added to the selection), and will end to the one you click with SHIFT+CTRL (or between the last two selected parts if you use the context menu or the shortcut key method).

Selection by name for Brick only

Finally, you can also select parts based on their name. This is very convenient if you want to select all the baseplate of a certain type for example. To do that you need to use the Find and Replace window. Read the Searching and Replacing Parts page for more details.
