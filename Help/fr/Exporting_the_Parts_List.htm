Exporter la liste de pièces


Exportation

Exporter une liste de pièces, est une option importante lorsque vous avez besoin d'une liste de vérification ou encore une liste d'achat. Cette option peut être trouvé dans le menu  Vue > Liste des Briques... .

Il est possible de séparer cette liste par couche en cochant la boite de sélection dans le bas de la fenêtre.

Pour sauvegarder la liste appuyez sur le bouton "Exporter...". Vous pouvez sauvegarder la liste dans un fichier texte ordinaire ou html si vous voulez inclure les images des pièces incluses dans la liste. Les liens internet des images seront inclus ce qui en fera une liste portable (vous pouvez l'envoyer à vos amis), une connexion internet est requise pour afficher les images correctement.

Impression

Il est possible d'imprimer la liste de pièces en ouvrant le fichier .txt ou .html dans lequel vous avez exporté la liste en utilisant le bouton "Exporter...", puis utiliser l'option d'impression du logiciel utilisé pour ouvrir le fichier.
