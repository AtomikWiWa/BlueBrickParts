Utiliser les textes

Les calques de texte vous permettent d'ajouter n'importe quel type d'annotation sur votre plan. Pour ajouter un texte sur un calque de texte, il suffit de double-cliquer sur le plan, une fenêtre s'ouvrira dans laquelle vous pourrez modifier votre texte, et après avoir cliqué sur le bouton Ok, le texte sera ajouté à l'endroit où vous avez double-cliqué. Plus tard, si vous voulez modifier un texte, double-cliquez à nouveau sur celui-ci, cela fera apparaître à nouveau la fenêtre pour éditer le texte.

Vous pouvez taper votre texte sur plusieurs lignes dans la boîte d'édition 1, et pour chaque texte, vous avez un contrôle total sur la Police, la Taille, le Style (gras, italique, etc...) 2, la couleur 3 et l'alignement (gauche, centre, droite) 4. Cependant, vous ne pouvez pas régler la transparence de chaque texte indépendamment mais vous pouvez modifier la transparence du calque de texte en double-cliquant sur le panneau du calque de texte. Donc si vous voulez avoir des transparences différentes pour des textes différents, il suffit d'utiliser des calques de texte différents. Les propriétés par défaut du texte peuvent être définies dans les Préférences d'apparence, ou comme pour les mesures, vous pouvez faire un clic droit sur un texte et choisir  Utiliser comme modèle .

Les textes peuvent être tournés selon n'importe quel angle, leur ordre de rendu peut être ajusté ("Envoyer à l'arrière" et "Ramener vers l'avant"), et bien sûr ils peuvent être déplacés mais ils ne sont pas affectés par l'aimantation de la grille. Ils peuvent également être copiés/collés et groupés comme les pièces.
