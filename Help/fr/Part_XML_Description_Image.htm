Définir un lien vers une image en ligne

Syntaxe: <ImageURL>...</ImageURL>
Valeur par défaut: Par défaut, BlueBrick construit cette URL:
http://media.peeron.com/ldraw/images/<ColorCode>/<PartNumber>.png
Description: Une adresse internet (URL) qui pointe vers une image de la pièce de telle manière que cette URL puisse être utilisée pour générer la liste des pièces au format HTML.

Quand vous exportez la liste des pièces de votre plan en HTML afin d'avoir un fichier portable, les images des pièces de la liste ne pointent pas vers votre disque local. Au lieu de cela BluBrick utilise une adresse Internet (URL) comme par exemple un lien vers les sites Peeron ou Bricklink. Si vous avez créé une pièce personalisée, vous pouvez mettre une image de celle-ci dans votre répertoire Brickshelf et préciser le lien Brickshelf de cette image dans ce tag XML.
