Propriétés du plan

En cliquant sur l'onglet  Propriétés  1 en bas à droite de la fenêtre BlueBrick, vous pouvez afficher et modifier les informations globales concernant votre plan. Toutes ces informations sont enregistrées dans le fichier BBM de BlueBrick, c'est pourquoi elles sont considérées comme les propriétés de votre plan.

Tout d'abord, en haut de la liste, vous pouvez modifier la couleur de fond de votre plan, en cliquant sur le bouton 2, ce qui fera apparaître une fenêtre de sélection de couleur. La couleur de fond peut également être éditée par le biais du menu  Edition > ; Couleur de fond du plan... .

Dans le champ  Auteur  3, vous pouvez saisir votre nom. Notez que si vous souhaitez que votre nom (et toutes les autres propriétés) soit affiché par défaut, vous devez créer un fichier modèle BBM, et définir ce fichier comme votre fichier modèle par défaut pour un nouveau plan. Voir les Préférences Générales pour plus de détails.

Dans le champ  LUG/LTC  4, vous pouvez sélectionner le "Lego User Group/Lego Train Club", pour lequel vous concevez ce plan. La boite combo contient une liste assez longue de LUG/LTC existants, mais vous pouvez taper n'importe quoi dans ce champ si vous ne trouvez pas votre LUG/LTC. N'hésitez pas à contacter Alban si vous souhaitez inclure votre LUG/LTC dans une prochaine version de BlueBrick.

Dans le champ  Evénement  5, vous pouvez sélectionner l'événement public pour lequel vous concevez cette mise en page. Ici encore, vous pouvez taper n'importe quoi si l'événement n'est pas répertorié ou si vous concevez simplement un plan personnel.

Dans le champ  Date  6, vous pouvez spécifier une date. Cette date peut être, par exemple, la date de l'événement, ou une date liée à la version du fichier. Par défaut, la date du jour courant sera utilisée lorsque vous créez un nouveau plan.

Les champs  Largeur  et  Hauteur  7 ne sont pas modifiables. Ils affichent en fait la taille actuelle de votre plan dans différentes unités (module = 3x3 plaque de base de 32x32 tenons = 96 tenons, tenons, mètre et pieds).

Dans le champ  Commentaire  8, vous pouvez ajouter le texte que vous voulez.

Enfin, si vous avez coché le menu  Vue > Filigrane , vous verrez un filigrane affiché par dessus votre plan dans le coin inférieur droit de votre plan 9. Le filigrane est composé en assemblant les propriétés Auteur, LUG/LTC, Evénement et Date, dans cet ordre. Actuellement, vous ne pouvez pas personnaliser la composition du filigrane.
