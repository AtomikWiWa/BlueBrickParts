Comment filtrer la bibliothèque de pièces

En bas de la bibliothèque de pièces, il y a une zone de texte 16 où vous pouvez taper n'importe quel type de mots clés pour filtrer les pièces affichées dans la bibliothèque de pièces. Le filtrage est effectué à la fois sur la description et la couleur de la pièce ; ce qui signifie que chaque pièce qui contient le mot-clé spécifié dans sa description ou son nom de couleur sera affichée, et les autres seront cachées. Effacez la zone de texte si vous souhaitez afficher la bibliothèque sans filtre.

Par défaut, lorsque la liste des pièces est filtrée, la couleur de fond de la bibliothèque de pièces devient légèrement rouge, mais vous pouvez personnaliser cette couleur dans les Prefereces de l'application, y compris en la mettant en blanc si vous ne voulez pas voir le changement de couleur.

Filtrage avancé

Vous pouvez utiliser plusieurs mots-clés séparés par un espace. Dans ce cas, toutes les pièces qui correspondent à tous les mots clés spécifiés (dans n'importe quel ordre) seront affichées. Maintenant, si vous voulez utiliser un espace à l'intérieur de votre mot-clé, vous devez inclure votre phrase de recherche entre guillemets. Par exemple, si vous voulez afficher toutes les plaques de base 32 x 32 vertes dans l'onglet "Baseplate" de la bibliothèque de pièces, vous devez taper : "32 x 32" green. Les doubles guillemets autour du texte "32 x 32" garantiront que ces trois mots ne seront pas séparés pendant le filtrage, et l'espace avant le mot "green" sera compris comme une séparation pour un deuxième mot-clé, par exemple, seuls les plaques de base qui sont à la fois vertes et ont une taille de 32x32 seront affichées.

Notez qu'avec l'exemple précédent "32 x 32" green une plaque de base SAND GREEN est également indiquée. Heureusement, il est également possible d'exclure les pièces qui correspondent à des mots-clés spécifiques en ajoutant un caractère moins - juste devant le mot-clé. Par exemple, si vous voulez exclure la plaque de base "sand green", vous pouvez utiliser "32 x 32" green -sand. Vous pouvez bien sûr utiliser uniquement des mots-clés exclusifs, par exemple si vous tapez -yellow vous afficherez toutes les pièces sauf les jaunes. Le caractère moins fonctionne également devant une phrase entre guillement, par exemple -"16 x 16" supprimera toutes les plaques de base 16x16. Ajouter un + devant un mot-clé aura le même effet que de ne pas l'ajouter.

Vous pouvez également filtrer sur la base des numéros de pièces. Pour ce faire, vous devez ajouter le caractère dièse # devant le mot-clé. Par exemple, en tapant #10 dans l'onglet "Town", la recherche sera plus facile pour trouver les bâtiments modulaires car leur numéro de set est souvent autour de 10 mille.

Filtrage par onglet ou pour tous les onglets

Il existe deux modes de filtrage et vous pouvez changer de mode en cliquant sur le bouton 17. Si vous appuyez sur ce bouton, son icône devient légèrement rouge, ce qui signifie que les mots clés que vous avez saisis dans la zone de texte seront utilisés pour tous les onglets de la bibliothèque de pièces. Dans ce cas, même si vous changez d'onglet, vous verrez toujours les mêmes mots clés de filtrage. Au contraire, en décochant le bouton 17, vous pourrez avoir des mots clés différents pour chaque onglet, et ils seront mémorisés lorsque vous changerez d'onglet.

Le mode global est utile si vous recherchez une pièce dont vous êtes assez sûr de la description, mais dont vous ignorez l'onglet dans lequel elle est stockée ; alors que le mode individuel est plus utile lorsque vous voulez restreindre certains onglets avec les pièces que vous utilisez le plus souvent. Par exemple, si vous utilisez principalement des rails 9V et des plaques de base standard, vous pouvez utiliser le mot-clé "32 x 32" dans l'onglet "Baseplate" et le mot-clé 9V dans l'onglet "Track".

Les mots clés sont automatiquement enregistrés lorsque vous quittez l'application, et rechargés à la prochaine session de Bluebrick.
