Utiliser un Budget

Comment créer un nouveau Budget

Si vous ne savez pas ce qu'est un budget dans BlueBrick, lisez d'abord la section Budget de pièces pour une description détaillée. Pour créer un nouveau budget à partir de zéro, choisissez le menu  Budget > Nouveau . La barre de titre de l'application sera mise à jour pour afficher le nom de votre nouveau budget qui est "MonBudget.bbb" par défaut. Bien entendu, lorsque vous créez un nouveau budget, il est totalement vide, ce qui signifie que la valeur maximale de toutes les pièces est soit "?" soit "0" selon la préférence que vous avez définie dans le menu Préférence de la bibliothèque de pièces, et par défaut, elle est "?". Ainsi, à partir de là, vous pouvez éditer la quantité de budget de chaque pièce de la bibliothèque (lire ci-dessous pour savoir comment faire) et enfin enregistrer votre budget dans un fichier, afin de le recharger plus tard (veuillez lire la section Enregistrer et ouvrir un budget pour plus de détails).

Si vous ne voulez plus utiliser ce budget, vous pouvez le fermer en choisissant le menu  Budget > Fermer . Bien entendu, si vous avez modifié votre budget, une boîte de dialogue apparaîtra pour vous demander si vous souhaitez l'enregistrer avant de le fermer. Lorsque votre budget présente des modifications non sauvegardées, une petite étoile apparaîtra à côté du nom du budget dans la barre de titre.

Comment modifier votre budget

Pour modifier votre budget, assurez-vous d'abord que vous affichez toutes les pièces de la bibliothèque, c'est-à-dire que le menu  Budget > Afficher uniquement les pièces budgétées  n'est pas coché, et que vous affichez les quantités de budget i.e. que le menu  Budget > Afficher les budgets  est coché (il s'agit des paramètres par défaut, mais lisez ci-dessous pour plus de détails sur ces options). Si c'est le cas, vous devriez voir deux nombres séparés par une barre oblique sous chaque pièce dans la bibliothèque des pièces. Le premier chiffre correspond au nombre réel de cette pièce dans votre carte actuelle, et le second est votre quantité de budget pour cette pièce. Vous pouvez inverser le compte dans la Préférence de la bibliothèque de pièces pour afficher la quantité disponible restante de la pièce au lieu du compte réel sur le plan.

Pour modifier cette quantité de budget, il suffit de cliquer dessus (cliquez vraiment sur le chiffre, pas sur la pièce). Vous pouvez également cliquer avec le bouton droit de la souris sur la pièce et choisir  Modifier le budget de cette pièce  dans le menu contextuel (cette méthode fonctionne également lorsque les quantité de budget sont cachés). Ensuite, la quantité de budget est affiché seul et devient modifiable. Vous pouvez maintenant taper n'importe quel nombre et appuyer sur la touche Entrée pour valider (ou cliquer ailleurs). Si vous entrez une valeur non valide, telle qu'une valeur négative, une valeur non entière, ou un texte qui n'est pas un entier, ou une entrée vide, alors votre entrée est ignorée et le budget pour cette pièce est supprimé. C'est la façon normale de supprimer une valeur de budget d'une pièce.

Par défaut, si un budget n'est pas fixé pour une pièce, un point d'interrogation sera affiché. Cela signifie que vous pouvez avoir un nombre infini de cette pièce sur votre plan, et BlueBrick ne vous avertira pas par rapport au nombre de cette pièce sur votre plan. Mais vous pouvez changer ce comportement, en demandant à BlueBrick de considérer toutes les pièces sans budget défini comme ayant un budget de 0. Dans ce cas, BlueBrick vous avertira ou vous empêchera d'ajouter toute pièce dont le budget n'est pas défini. Pour choisir ce comportement, vous devez aller dans le menu  Edition > Préférences...  et choisir l'onglet Bibliothèque de pièces (voir la section Préférence de la bibliothèque de pièces pour plus de détails).

La modification du budget de toutes les pièces de la bibliothèque est un travail fastidieux, aussi la plupart du temps votre budget ne concerne que quelques pièces de la bibliothèque. Lorsque vous utilisez un budget, la plupart du temps, vous voulez juste suivre le budget de certaines pièces critiques comme vos rails et vos plaques de base de route. Dans ce cas, choisir un budget infini pour les pièces non budgétées est ce que vous recherchez. Cependant, si vous avez une très petite collection LEGO® et que vous voulez être sûr de pouvoir construire le diorama que vous avez conçu dans BlueBrick, vous pouvez prendre le temps de définir le budget exact de votre collection, et choisir de considérer toutes les autres pièces avec un budget nul.

Options du budget

Une fois que vous avez fixé les quantités de votre budget, BlueBrick peut les utiliser de manière plus ou moins restrictive, selon ce que vous recherchez. Vous voulez peut-être simplement savoir si vous dépassez un budget, mais vous voulez quand même pouvoir construire parce que vous comptez demander à un ami de vous prêter les pièces manquantes ? Ou peut-être voulez-vous que BlueBrick soit très strict avec vous et vous empêche de dépasser un budget ? Vous pouvez facilement passer à tout moment de l'un à l'autre de ces deux modes via le menu  Budget > Utiliser la limite de budget . Si cette option est cochée, BlueBrick vous empêchera d'ajouter plus de pièces que le nombre autorisé par votre budget. Cela peut se produire si vous essayez de faire glisser la pièce sur le plan, de faire un copier/coller ou une duplication, ou d'essayer de remplacer une autre pièce par celle-ci. Dans tous ces cas, BlueBrick émet un bip sonore et affiche un message d'avertissement que vous pouvez désactiver pour ne plus le voir plus tard. Donc si vous constatez que certaines pièces manquent lors de votre copier/coller, alors avant de signaler un bug, veuillez vous assurer que vous n'avez pas atteint la limite budgétaire pour ces pièces.

La limitation budgétaire n'est pas utilisée lorsque vous ouvrez un plan. Il peut donc arriver que certaines pièces dépassent votre budget si vous chargez une mise en page créée par quelqu'un d'autre. De même, si vous réduisez le budget d'une pièce, BlueBrick ne supprimera pas aléatoirement certaines de ces pièces dans votre plan pour atteindre votre nouveau budget, ceci est laissé à votre discrétion. Dans tous les cas, que la limitation du budget soit utilisée ou non, toute pièce qui dépasse le budget aura sa quantité de budget affiché sur fond rouge.

L'option  Budget > Afficher les budgets  vous permet d'afficher ou de masquer les quantités de budget sous les pièces dans la bibliothèque des pièces. Si vous préférez maximiser l'espace pour l'affichage des pièces, vous pouvez cacher les chiffres. La limitation du budget continuera à fonctionner même si les chiffres sont cachés. Vous pouvez toujours modifier une quantité de budget même si elle est masquée, en cliquant avec le bouton droit de la souris sur la pièce et en choisissant  Modifier le budget de cette pièce .

Enfin, l'option  Budget > Afficher seulement les pièces budgétées  est assez radicale, car elle masque toutes les pièces sauf celles qui ont une quantité de budget définie. Comme cette option fera disparaître de nombreuses pièces de la bibliothèque, la couleur de fond de la bibliothèque de pièces devient jaune, mais vous pouvez changer cette couleur dans les Préférences de la bibliothèque de pièces. Cette option peut être utile pour ceux qui ont créé un budget pour leur collection LEGO® et qui veulent construire leur plan uniquement avec ces pièces : en montrant uniquement les pièces de votre collection, vous faciliterez la recherche de pièces dans la bibliothèque.

Ces trois options peuvent également être modifiées via le menu contextuel de la bibliothèque de pièces en cliquant sur le bouton droit de la souris.
