Afficher et exporter la liste de pièces

Vous pouvez afficher la liste de toutes les pièces utilisées dans le plan actuellement ouvert, en cliquant sur l'onglet  Liste de pièces  1 dans le coin inférieur droit de la fenêtre BlueBrick. Notez que si vous avez un fichier de budget ouvert, comme dans la capture d'écran ci-dessous, vous verrez plus de colonnes dans la liste (voir la section Budget de pièces pour plus de détails concernant les budgets).

Informations affichées

Voici une description de toutes les colonnes :

  • La colonne 2 affiche le numéro de pièce avec une petite icône de la pièce.
  • La colonne 3 affiche la quantité utilisée sur le plan actuel pour chaque pièce. Si vous supprimez toutes les pièces d'un type, cette pièce sera supprimée de la liste.
  • La colonne 4 affiche le décompte budgété de la pièce (si vous avez un budget ouvert, sinon la colonne est cachée). Si la pièce n'est pas budgétée, le mot "Hors-Budget" est écrit à la place.
  • La colonne 5 affiche la quantité de pièces dépassant le budget (si vous avez un budget ouvert, sinon la colonne est cachée), c'est-à-dire la quantité de pièces qui seraient manquantes si vous essayez de construire le plan ouvert avec le budget ouvert. Les pièces non budgétées sont également comptées comme manquantes.
  • La colonne 6 affiche un pourcentage d'utilisation des pièces, en fonction du budget ouvert (si vous avez un budget ouvert, sinon la colonne est cachée). Ce pourcentage peut dépasser 100% si le plan comporte plus de pièces que le nombre autorisé dans le budget. Si la pièce n'est pas budgétée, le mot "Hors-Budget" est écrit à la place.
  • La colonne 7 affiche la couleur de la pièce.
  • La colonne 8 affiche la description de la pièce.

Enfin, à la fin de chaque liste de calques, il y a une ligne qui affiche le total de chaque colonne.

Vous pouvez diviser la liste par calque en cochant la case 9 en bas de la fenêtre. Si cette case n'est pas cochée, toutes les pièces du même type sont comptées ensemble, même si elles sont placées dans des calques différents. Vous pouvez également inclure les pièces cachées (c'est-à-dire les pièces qui sont placées sur des calques cachés) en cochant la case 10 au bas de la fenêtre. Si cette case n'est pas cochée, les calques cachés sont ignorés dans le décompte.

Vous pouvez réorganiser les pièces de la liste en cliquant sur l'en-tête de chaque colonne. De cette façon, vous pouvez afficher toutes les pièces manquantes en haut de la liste par exemple. Le petit triangle noir à côté du nom de la colonne indique quelle colonne est utilisée pour le tri et dans quel ordre (ascendant ou descendant). Vous pouvez cliquer à nouveau sur un en-tête de colonne pour inverser son ordre de tri.

Exportation

Exporter la liste des pièces, peut être pratique pour créer une liste d'achats, ou pour la partager avec d'autres participants à un événement. Cette fonctionnalité se trouve via le menu  Fichier > Exporter la liste de Pièces... . Une boîte de dialogue d'enregistrement de fichier apparaîtra, dans laquelle vous pourrez spécifier un nom de fichier pour exporter la liste. Vous pouvez enregistrer le fichier sous forme de fichier texte simple, de fichier CSV ou de fichier HTML. Voici les différents avantages entre ces formats :

  • Format texte : Le format texte est un format rapide, sale et léger. Il peut être pratique si vous souhaitez le copier/coller dans un e-mail, car il est formaté pour être humainement lisible.
  • Format CSV : CSV signifie Comma Separated Values, et est un format lisible par des logiciels de tableur tels que Microsoft Excel, ou OpenOffice Calc. Vous pouvez utiliser ce format si vous voulez faire des statistiques sur la liste des pièces dans un tableur.
  • Format HTML : Ce format est agréablement formaté et comprend les images des pièces. Ces images sont référencées sur internet, ce qui rend le fichier portable (vous pouvez l'envoyer à vos amis), mais nécessite bien sûr une connexion internet pour être affiché correctement. Les références des images sont définies dans le fichier Définir un lien vers une image en ligne, ce qui peut entraîner des liens rompus si ce fichier n'est pas correctement configuré.

Impression

Il est possible d'imprimer la liste de pièces en ouvrant le fichier .txt, .csv ou .html dans lequel vous avez exporté la liste, puis utiliser l'option d'impression du logiciel utilisé pour ouvrir le fichier.
