Chercher et remplacer des pièces


On peut accéder à la fonction Rechercher et Remplacer... via le menu  Edition > Rechercher et remplacer...  ou en appuyant sur CTRL+F.

Le champ Recherche peut être utilisé pour trouver toutes les pièces de votre choix dans les calques que vous sélectionnez. Pour ce faire, il suffit de sélectionner la pièce dans la liste déroulante du haut, de choisir les calques dans lesquelles vous souhaitez effectuer la recherche et d'appuyer sur "Sélectionner tout".

Le champ Remplacer est très utile si vous voulez changer une sélection de pièces, ou toutes les pièces d'un type spécifique dans certaines calques, en quelque chose d'autre. De bons exemples sont le changement d'une voie 9V en voie 12V, ou si vous voulez changer le type/version d'une plaque de route spécifique que vous avez utilisée dans votre disposition.

La section Où ? vous permet de préciser où chercher les pièces. Si vous choisissez "Dans la sélection courante", la recherche ne portera que sur la pièce de la sélection courante, donc vous allez sous-sélectionner toutes les instances de cette pièce qui étaient dans la sélection (si vous cliquez sur "tout sélectionner") ou vous allez remplacer seulement les instances de cette pièce qui étaient dans la sélection (si vous cliquez sur "tout remplacer") en laissant les autres instances dans le calque courant. De plus, si vous choisissez "Dans la sélection courante", la liste déroulante "Rechercher" ne contient que les pièces de la sélection, ce qui est généralement beaucoup moins que la bibliothèque entière. Par conséquent, une façon rapide et pratique de trouver ou de remplacer une pièce spécifique de votre plan est la suivante : cliquez d'abord sur cette pièce dans votre plan pour la sélectionner, puis appuyez sur CTRL+F pour ouvrir la fenêtre de recherche, puis modifiez les options "Où ?" pour étendre la recherche à tous les calques.

Si vous souhaitez étendre la recherche à un ou plusieurs calques, vous devez choisir l'option "Dans le(s) calque(s) suivant(s) :", qui activera la liste des calques de briques juste en dessous. Vous pouvez alors choisir le calque dans lequel vous souhaitez effectuer la recherche. Cependant, sachez que si vous choisissez plusieurs calques, vous ne pourrez pas sélectionner toutes les pièces (parce que BlueBrick ne supporte pas encore la sélection multi-calque, cela pourrait être ouvert dans une future version de BlueBrick), mais vous pouvez faire un remplacement dans plusieurs calques.
