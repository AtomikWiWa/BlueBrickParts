Comment configurer les racourcis clavier

Le bouton 1 vous permet de restaurer toutes les préférences aux valeurs par défaut, comme lorsque vous avez installé BlueBrick pour la première fois. Le bouton 2 vous permet de ne restaurer que les préférences affichées sur la catégorie visible actuellement.

Ajouter des touches de raccourci peut être fait en utilisant les quatre menus déroulants au bas de la liste des raccourcis. Choisissez chacune des touches de votre clavier à laquelle vous souhaitez attribuer la fonction 3, puis la fonction que cette touche est censée exécuter 4, la pièce ou le numéro de set qu'elle utilisera 5 si nécessaire, et le type de connexion qui doit être utilisé pendant l'action 6 si nécessaire. Pour compléter l'ajout, appuyez sur le bouton "Ajouter raccourci" 7.

Notez qu'un raccourci ne peut pas être édité une fois qu'il a été créé. Vous devez supprimer le raccourci que vous souhaitez modifier et en créer un nouveau. Pour supprimer un raccourci, sélectionnez-le et appuyez sur le bouton "Supprimer raccourci" 8. Vous pouvez également supprimer ou réassigner les raccourcis par défaut si vous ne les aimez pas.

Ces changements seront permanents une fois que vous aurez appuyé sur le bouton "OK". Si vous appuyez sur "Cancel", les changements seront annulés.
