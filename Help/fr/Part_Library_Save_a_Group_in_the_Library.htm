Comment sauvegarder un groupe dans la bibliothèque

Bluebrick dispose d'une fonction pratique permettant de regrouper plusieurs pièces (lire la section Groupement de pièces pour plus de détails). C'est en effet pratique pour déplacer un arrangement complexe de pièces sur votre plan, mais vous avez peut-être envie d'enregistrer ce groupe dans la bibliothèque de pièces pour une utilisation ultérieure dans un autre plan. Heureusement, cela est possible : vous devez d'abord sélectionner un groupe ou plusieurs pièces de votre plan et choisir le menu  Fichier > ; Sauvegarder la sélection dans la bibliothèque...  (si aucune ou une seule partie est sélectionnée, le menu sera désactivé) ; cela affichera cette fenêtre :

Vous devez alors donner un nom unique à votre groupe dans la zone de texte 1. Ce nom unique sera utilisé comme identifiant de la pièce et aussi comme nom du fichier créé pour enregistrer le groupe sur le disque. C'est pourquoi vous ne pouvez pas utiliser de caractères spéciaux comme le point d'interrogation "?". La zone de texte restera rouge tant que le nom n'est pas valide. BlueBrick vous propose ".SET" comme extension, selon sa propre convention, mais vous pouvez choisir une autre extension.

Le fichier XML contenant la description de votre groupe sera enregistré dans le dossier "parts\Custom" du dossier d'installation de BlueBrick. Si ce dossier "Custom" n'existe pas, il sera créé. Cependant, vous pouvez déplacer ce fichier dans un autre dossier avec l'explorateur de fichiers de votre système d'exploitation, puis recharger la bibliothèque de pièces.

.

Vous pouvez également spécifier une clé de tri pour votre groupe dans la zone de texte 2, par exemple pour contrôler l'endroit où il est répertorié dans la bibliothèque de pièces (lire la section Clé de tri pour plus de détails).

La case à cocher 3 vous permet de choisir si vous voulez que votre groupe soit dégroupable ou non. La plupart des groupes devraient être dégroupables pour un maximum de flexibilité pour l'utilisateur, cependant si vous créez un ensemble en deux pièces reliées par une charnière, vous ne devriez pas permettre à l'utilisateur de séparer cet ensemble.

La zone de texte Auteur 4 affichera votre nom si vous l'avez défini dans les Propriétés du plan, sinon juste "Auteur inconnu" par défaut. Le but est de vous donner du crédit au cas où vous décideriez de partager votre groupe sur Internet. Tapez ce que vous voulez ou laissez le champ vide si vous n'êtes pas intéressé par la célébrité et la gloire.

Vous pouvez ensuite taper une description de votre groupe dans différentes langues : Vous devez d'abord choisir le code de la langue dans la liste déroulante 5 et ensuite taper la description dans cette langue dans la zone de texte 6. Vous pouvez bien sûr saisir une description en plusieurs langues.

La zone de texte URL de l'image 7 est utile si vous créez un set officiel dont l'image peut être trouvée sur internet, alors veuillez copier/coller le lien ici tel que BlueBrick peut l'utiliser lorsque vous exportez la liste de pièces. L'URL de la pièce 8 n'est pas encore utilisée dans cette version de BlueBrick.

Une fois que vous avez fini de remplir le formulaire et que vous avez cliqué sur Ok, la pièce sera enregistrée sur le disque et l'onglet "Custom" sera sélectionné dans la bibliothèque de pièces pour afficher votre nouvelle pièce. La nouvelle partie peut être utilisée immédiatement en la faisant glisser sur votre carte. Attention, si vous prévoyez de partager votre plan, vous devrez également partager vos nouvelles pièces avec vos amis, sinon ils verront simplement une croix rouge à la place de vos pièces lors de l'ouverture du fichier BBM que vous leur avez envoyé.
