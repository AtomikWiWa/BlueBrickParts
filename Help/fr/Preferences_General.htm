Préférences Générales

Le bouton 1 vous permet de restaurer toutes les préférences aux valeurs par défaut, comme lorsque vous avez installé BlueBrick pour la première fois. Le bouton 2 vous permet de ne restaurer que les préférences affichées sur la catégorie visible actuellement.

La Langue du logiciel 3 peut être changée dans le coin supérieur gauche. Le changement sera effectif pour presque tout les textes du logiciel. Vous devez cependant redémarrer BlueBrick afin que votre choix prenne effet. Les items qui ne sont pas affectés sont les onglets au dessus du panneau de pièces et les fenêtres qui dépendent de la langue de votre système d'exploitation (par exemple la palette de sélection des couleurs).

Les Notifications 4 listent quelques messages d'avertissement qui peuvent apparaître pendant l'exécution de BlueBrick. Comme ces messages d'avertissement peuvent être répétitifs, vous pouvez les désactiver en cochant une case "oublier". Par défaut, toutes les notifications d'avertissement sont activées. Vous pouvez rétablir les messages de notification que vous avez choisi d'ignorer, ou désactiver quelques notifications à l'avance.

La liste de Fichiers Récents 5 est de 4 par défaut. Elle peut être augmentée jusqu’à un maximum de 20. Vous pouvez vider la liste en utilisant le bouton "Vider la liste des fichiers récents" à n'importe quel moment. Notez que la liste une fois éffacée ne peut être rétablie.

La longueur de la liste Défaire/Refaire 6 peut être ajustée. Le maximum d'actions enregistrées est de 1000. Le nombre maximum d'actions affichées est de 50. Prenez note que plus grande est le nombre d'opérations enregistrées, plus grande est la quantité de mémoire requise par le logiciel.

Le champ Configuration par défault des nouveaux plans 7 vous permet de spécifier un fichier BBM qui doit être utilisé comme modèle chaque fois que vous créez un nouveau plan. Pour ce faire, créez d'abord un fichier BBM avec les options que vous préférez, comme le nombre et le type de calques, la couleur de fond, votre nom en tant qu'auteur, le logo de votre LUG dans un coin, etc... et enregistrez-le quelque part sur votre disque dur. Ensuite, indiquez le chemin d'accès de ce fichier dans ce champ. Maintenant, chaque fois que vous utilisez la fonction  Fichier > Nouveau , un nouveau plan sera créé en utilisant ce fichier comme point de départ.

L'ajustement des Performances 8 peut être important pour les ordinateurs moins récents. L'option "Rapide" est sélectionné par défaut. Vous pouvez changer celui-ci pour "Moins de mémoire" si vous notez que votre système devient plus lent ou instable durant l'utilisation. La majorité des ordinateurs devraient être capable de fonctionner en mode "Equilibré" ou "Rapide".
