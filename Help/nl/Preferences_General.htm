Algemene opties

Met de knop 1 kun je alle voorkeuren terugzetten naar hun standaardwaarden, zoals toen je BlueBrick voor de eerste keer installeerde. Met de knop 2 kun je alleen de voorkeuren herstellen die in de huidige zichtbare categorie worden weergegeven.

De Programma taal 3 kan worden veranderd in de linkerbovenhoek. De wijziging is van invloed op bijna alle tekst gebeurtenissen in het programma. Je moet BlueBrick opnieuw opstarten om de wijzigingen door te voeren. Punten die niet worden beïnvloed door de verandering zijn de tabbladnamen van het onderdelen venster, en dialogen die rechtstreeks verband houden met de taal van je OS (bijvoorbeeld kleur selectie palet).

The Notifications 4 list some warning messages that can popup during the execution of BlueBrick. Since those warning messages can be repetitive, you can disable them by checking a "forget" checkbox. By default all the warning notifications are enabled. You can re-enable the notification messages you have chosen to ignore, or disable some notifications upfront.

De recente Bestanden Lijst 5 is ingesteld op 4 als standaard. 20 is het maximum. Je kunt de lijst met de "Clear Recent File List" knop op elk moment legen. Merk op dat deze actie niet ongedaan kan worden gemaakt.

De Undo / Redo 6 lijsten kunnen worden aangepast. Het maximum aantal undo's opgenomen is 1000. Het weergegeven maximum aantal is 50. Merk op dat hoe groter het aantal undo's opgenomen, des te meer het geheugenaanvraag er is.

The New Map Template field 7 let you specify a BBM file that should be used as a template whenever you create a new layout. To do that, first create a BBM file with the options you prefer, like the number and type of layers, the background color, your name as author, the logo of your LUG in one corner, etc... and save it somewhere on your hard drive. Then specify the path of that file in this field. Now on, every time you use the  File > New  feature, a new layout will be created by using this file as the starting point.

De Performance-instellingen 8 kunnen belangrijk zijn voor oudere computers. De "Sneller" instelling wordt gekozen als standaard. Je kunt kiezen voor de "minder geheugen" optie als je merkt dat je systeem instabiel wordt tijdens het gebruik. De meeste computers moeten kunnen draaien in de "evenwichtige" of "sneller" modi.
