Auswahl von Teilen

Standard-Auswahl

Die allgemeine Auswahl funktioniert für die Ebenen "Stein", "Text" und "Lineal" auf die gleiche Weise. Sie können einzelne Teile auswählen, indem Sie sie anklicken, oder mehrere Teile, indem Sie die Maus über die gewünschten Teile ziehen (und ein Auswahlrechteck bilden). Die Art und Weise, wie ausgewählte Teile hervorgehoben werden, kann unter der Registerkarte "Erscheinungsbild" in den BlueBrick-Voreinstellungen angepasst werden.

Die Auswahl mehrerer Teile kann durch Halten der STRG-Taste (standardmäßig) und Anklicken der Teile mit der linken Maustaste erfolgen. Wenn Sie die STRG-Taste gedrückt halten und auf ein bereits ausgewähltes Teil klicken, wird es abgewählt.

Sie können auch eine Gruppe von Teilen auswählen, indem Sie die linke Maustaste gedrückt halten und die Maus so ziehen, dass sich die gewünschten Teile innerhalb des rechteckigen Auswahlrahmens befinden. Sie können mehrere rechteckige Auswahlen treffen, indem Sie beim Ziehen weiterer Auswahlboxen die STRG-Taste (standardmäßig) gedrückt halten. Wenn Sie die STRG-Taste nicht gedrückt halten, verschwindet das vorherige Auswahlfeld. Wenn sich zwei Auswahlrahmen überschneiden und nicht die gleichen Teile enthalten, werden alle ausgewählten Teile zu einem großen Auswahlrahmen zusammengeführt. Wenn Sie eine Auswahlbox über bereits ausgewählte Teile ziehen, werden diese abgewählt.

Pfadauswahl nur für Brick

Sie können auch eine Reihe von zusammenhängenden Teilen auswählen, wie z. B. ein Stück Schienen oder Straßengrundplatten, indem Sie zuerst ein Teil auswählen, dann SHIFT+STRG (standardmäßig) gedrückt halten und auf ein zweites Teil klicken. Alle Teile zwischen den beiden ausgewählten Teilen werden auf dem kürzesten Weg ausgewählt. Wenn kein Weg zwischen den beiden zunächst ausgewählten Teilen gefunden wird, passiert nichts. Alternativ können Sie zuerst die beiden Extremitäten auswählen, dann mit der rechten Maustaste klicken und im Kontextmenü Select Path wählen oder STRG+P drücken.

Beachten Sie, dass der Pfad immer mit dem zuletzt ausgewählten Teil beginnt (d. h. dem zuletzt zur Auswahl hinzugefügten Teil) und an dem Teil endet, das Sie mit SHIFT+STRG anklicken (oder zwischen den letzten beiden ausgewählten Teilen, wenn Sie das Kontextmenü oder die Tastenkombinationen-Methode verwenden).

Auswahl nach Namen nur für Brick

Schließlich können Sie Teile auch anhand ihres Namens auswählen. Dies ist sehr praktisch, wenn Sie z. B. alle Grundplatten eines bestimmten Typs auswählen möchten. Dazu müssen Sie das Fenster "Suchen und Ersetzen" verwenden. Lesen Sie die Seite Suchen und Ersetzen von Teilen für weitere Details.
