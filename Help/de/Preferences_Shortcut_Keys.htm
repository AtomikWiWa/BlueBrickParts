So passen Sie die Tastenkombinationen an

Mit der Schaltfläche 1 können Sie alle Einstellungen auf ihre Standardwerte zurücksetzen, wie bei der Erstinstallation von BlueBrick. Mit der Schaltfläche 2 können Sie nur die Einstellungen wiederherstellen, die in der aktuell sichtbaren Kategorie angezeigt werden.

Hinzufügen von Tastaturkürzeln ist mit Hilfe der vier Auswahlmenüs unterhalb der Tastaturkürzel-Liste (Menü Bearbeiten, Allgemeine Einstellungen...) möglich: Wähle zuerst die Taste aus 3, der eine Funktion zugeordnet werden soll, und dann die Funktion 4. Bei der Funktion "Teil hinzufügen" muss noch die Teil- oder Setnummer ausgewählt 5 und der Verbindungstyp 6 angegeben werden. Zum Abschluss muss nur noch die Schaltfläche "Tastenkürzel hinzufügen" 7 angeklickt werden.

Ein Tastaturkürzel kann nicht direkt geändert werden. Um einer Taste eine andere Funktion zuzuweisen, muss zunächst das Tastaturkürzel gelöscht werden. Danach kann der Taste eine neue Funktion zugewiesen werden.

Wird dann die Schaltfläche "Ok" angeklickt, werden die Standardeinstellungen wieder eingestellt.
