Bildschirmaufteilung

Der Bildschirm von BlueBrick besteht aus 3 Segmenten: das Gleisplan-Segment, das Teile-Segment und das Fenster Kartendaten. Die Benutzung dieser Segmente wird im Abschnitt Erstellung und Änderung von Gleisplänen detailiert beschrieben.

Das Gleisplan-Segment

Der größte Teil des Bildschirms wird durch das Gleisplan-Segment belegt. Standardmäßig besitzt dieses Segment einen blauen Hintergrund und ein weißes Raster. Innerhalb dieses Segmentes können die einzelnen Teile des Gleisplans einfach mit der Maus angeklickt und verschoben werden.

Das Teile-Segment

Das Teile-Segment befindet sich oben rechts und besteht aus mehreren Karteireitern mit unterschiedlichen Teilekategorien. Durch die Gliederung der Teile in verschiedene Kategorien sind die einzelnen Teile einfach zu finden.

Das Fenster Kartendaten

Dieser Bereich befindet sich in der rechten unteren Ecke und hat drei Registerkarten. Auf der ersten Registerkarte werden die Ebenen angezeigt, auf der zweiten Registerkarte die Verwendung der Teile und auf der letzten Registerkarte werden einige allgemeine Informationen zur Karte angezeigt. Die Ebenenfunktion ist besonders für kompliziertere Layouts nützlich. In dieser Registerkarte können Sie die verschiedenen Ebenen, die Sie benötigen, hinzufügen, entfernen und organisieren. Details finden Sie im Abschnitt Erstellen und Verwenden von Layern. Die Registerkarte "Teileverwendung" zeigt eine Liste aller Teile an, die in der Karte verwendet werden, und schließlich die prozentuale Verwendung im Vergleich zum Budget. Auf der letzten Registerkarte können Sie einige Informationen zu Ihrer Map hinzufügen.
