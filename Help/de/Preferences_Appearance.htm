Aussehen (Erscheinungsbild) einstellen

Mit der Schaltfläche 1 können Sie alle Einstellungen auf ihre Standardwerte zurücksetzen, wie bei der Erstinstallation von BlueBrick. Mit der Schaltfläche 2 können Sie nur die Einstellungen wiederherstellen, die in der aktuell sichtbaren Kategorie angezeigt werden.

Das Standard- Farbschema 3 ist "Blaupause". Es kann auch "Pergament" oder "Klassisch" ausgewählt werden. Benutzerdefinierte Farbschemata können durch Auswahl "Erweitert" und Anklicken der farbigen Kästchen hinter "Hintergrundfarbe", "Standard Rasterfarbe" und "Standard Unterrasterfarbe" erstellt werden.

Below, you can adjust the highlight percentage for the selected parts and for the single part you grab in a group of selected parts. You can also set the default color and size of the item hull (this apply to parts, texts and rulers).
Das Ergebnis der Farbauswahl wird im Vorschaufenster angezeigt. Mit Anklicken des Buttons "OK".

You can also set the default text format 4 (font style and color) which will be used each time you create a new text on your map. This settings also affect the dimension displayed by the rulers.

Similarly you can set the default settings for each new grid layer 5 and each new area layer 6. To tune the default ruler style see the Edition Preference section.
