Allgemeine Einstellungen

Die Sprache 3 kann oben links geändert werden. Diese Änderung betrifft nahezu das gesamte Programm. Nach der Änderung muss BlueBrick neu gestartet werden, damit BlueBrick in der gewählten Sprache erscheint. Unverändert bleiben lediglich die Namen der Karteireiter der Teileliste im Fenster oben rechts. Auch Fenster, die zur im Betriebssystem eingestellten Sprache verknüpft sind, wie z.B. die Farbtabelle, bleiben unverändert.

The Notifications 4 list some warning messages that can popup during the execution of BlueBrick. Since those warning messages can be repetitive, you can disable them by checking a "forget" checkbox. By default all the warning notifications are enabled. You can re-enable the notification messages you have chosen to ignore, or disable some notifications upfront.

Die Anzahl der zuletzt benutzten Dateien 5 ist standardmäßig auf 4 gesetzt, 20 ist maximal möglich. Diese Liste kann mit der Schaltfläche "Liste der zuletzt benutzten Dateien leeren" jederzeit gelöscht werden. Dieses Löschen kann nicht rückgängig gemacht werden.

Die Liste Rückgängig / Wiederherstellen 6 kann ebenfalls eingestellt werden. Maximal 1000 Schritte können gespeichert und damit rückgängig gemacht werden. Maximal 50 Schritte können angezeigt werden. Je größer die Anzahl der gespeicherten Schritte ist, desto mehr Speicherplatz benötigt BlueBrick.

The New Map Template field 7 let you specify a BBM file that should be used as a template whenever you create a new layout. To do that, first create a BBM file with the options you prefer, like the number and type of layers, the background color, your name as author, the logo of your LUG in one corner, etc... and save it somewhere on your hard drive. Then specify the path of that file in this field. Now on, every time you use the  File > New  feature, a new layout will be created by using this file as the starting point.

Die Leistungs-Einstellungen 8 könnten für ältere Computer wichtig sein. Voreingestellt ist "Schneller". Wenn BlueBrick während der Benutzung instabil wird, sollte die Einstellung "weniger Speicher" benutzt werden. Die meisten Computer arbeiten auch mit den Einstellungen "ausgeglichen" oder "schneller" einwandfrei.
